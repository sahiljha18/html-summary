Introduction of webdevelopment/internet workings

  1. What is the Web / Internet Difference between the Internet (global network of computers) and the Web (services/applications using the Internet).
    • How devices are connected — using IP addresses, routers, DNS to resolve domain names
    • How data travels: packets, protocols like TCP, UDP, HTTP

  2. Client-Server Model

    • Client (browser) sends a request to server; server processes request, possibly uses database, then responds.
    • HTTP / HTTPS as the protocol used in web communication: request methods (GET, POST, etc.), status codes.

  3. Web Development Roles and Types

    • Frontend development: what users see → layout, styles, interactivity. Technologies like HTML, CSS, JavaScript.
    • Backend development: what happens behind scenes → server logic, databases, APIs.
    • Full-stack developers: people who work on both front and back ends

  4. Core Technologies

    • HTML (HyperText Markup Language): structure of web pages — headings, paragraphs, links, images, etc.
    • CSS (Cascading Style Sheets): styling — colors, layouts, responsive design, formatting.
    • JavaScript: making pages interactive — handling user inputs, DOM manipulation, events.

  5. Tools & Setup

    • Choosing and using a code editor (VSCode, Sublime, etc.)
    • Using a browser to test / inspect code.
    • Simple “development environment” setup (folder structure, saving files, opening them in browser).

  6. Putting Things Together

    • Creating a simple webpage: write HTML file, style with CSS, view it in browser
    • Understanding basic structure:
    • The edit-refresh cycle: how you make changes, save file, refresh browser to see changes.

  7. Other Important Concepts

    • Responsiveness (making websites look good on devices of different sizes).
    • Accessibility basics — making sites usable by people with different abilities.:
    • Basics of deploying or publishing (uploading to a server or hosting).

  8. Why It Matters / Career Perspective

    • High demand for web developers.
    • Wide opportunity: many simple websites, web apps, e-commerce, mobile web
    • Continuously evolving field: staying updated with new technologies matters.
