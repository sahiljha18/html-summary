  1. class attributes

    • Used to group multiple elements under the same name.
    • Allows you to apply CSS styles or JavaScript actions to many elements at once.
    • You can use the same class name on multiple elements
    • in representing this we use (.)in css external sheet.

  2. id attribute

    • A unique identifier for one element (must not repeat on a page).
    • Useful for specific styling or linking directly (anchors).
    • Often used in JavaScript to select a single element.
    • it is denoted by (#)

  3. div tag

    • A (generic container short for division).
    • Groups elements together for layout and styling.
    • Has no special meaning â€” mostly used with class or id.

  4. header tag

    • Represents the top section of a page or section.
    • Usually contains: logo, navigation menu, page title.
    • Improves SEO and semantic meaning

  5. footer tag

    • Represents the bottom section of a page or section
    • Usually contains: contact info, copyright, links, social media.

  6. main tag

    • Defines the main content of the page
    • Only one <.main> per page
    • Should contain the unique content of that page (not repeated across pages like navbars, headers, or footers).
